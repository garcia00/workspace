= Hemopa Digital


A simple project to get you started with https://github.com/adminfaces[AdminFaces^].

== Building

This project uses https://maven.apache.org/[Maven^] for the build:

----
mvn clean package
----

== Running

Just deploy `hemopa-digital.war`.
